## Screenshots (you capture on device/emulator)

Play requires at least **2** phone screenshots (16:9 or 9:16). Suggested set:

1. **Today** — tomorrow’s plans hub  
2. **Plan detail** — structured lesson fields  
3. **Timetable** — weekly periods  
4. **AI Assist** — chat / quick tools  
5. **Settings** — backup + privacy links  

Save phone screenshots under `store/graphics/phone/` when ready (not committed until you add them).

### How to capture

```powershell
cd d:\EduFrame
flutter run --release
# Then device screenshot, or Android Studio → emulator camera icon
```
